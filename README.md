# 我是教皇 - 瘟疫抉择 (Papal Decisions: The Plague Era)

**"我是教皇 - 瘟疫抉择"** 是一款文字互动游戏，玩家将扮演一位中世纪的教皇，在黑死病肆虐的时代做出决定，影响民众的生死存亡。你的选择将改变历史，你会如何应对瘟疫的挑战？
